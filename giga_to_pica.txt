1.1 How many bytes are in a kilobyte?
In standard modern definition 1kb = 1000 bytes
However in older binary system 1Kb = 1024 bytes 
So our answer in Binary Tradition will be 1024 bytes.

1.2 How many megabytes are required to express a single gigabyte?
1 GB = 1024 megabytes

1.3 If memory in a computer is byte-addressable, meaning each address points to a single byte, how many addresses does 32 kilobytes of that memory have?
By using direct proportion 
23KB multiplied by 1024 bytes give us 32 768 bytes
